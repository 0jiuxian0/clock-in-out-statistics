# 打卡统计计算器

一个纯前端的加班统计工具，可以根据企业微信导出的打卡记录Excel文件，自动计算加班时长、补贴金额等信息。

## 功能特性

- 📊 **自动统计**：根据打卡记录自动计算加班时长
- 💰 **补贴预估**：根据加班规则自动计算补贴金额
- 📅 **工作日计算**：自动排除周末和法定节假日
- 🎨 **深色模式**：支持浅色/深色主题切换
- 📱 **响应式设计**：支持手机和电脑访问
- 💾 **数据持久化**：自动保存统计数据，刷新不丢失
- ⚙️ **灵活配置**：可手动排除特定日期的打卡记录

## 使用说明

### 本地开发

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 访问 `http://localhost:5173`

### 构建部署

1. 构建生产版本：
```bash
npm run build
```

2. 构建产物在 `dist` 目录，可以部署到：
   - GitHub Pages
   - 任何静态文件服务器

### 部署到 GitHub Pages

1. 在 GitHub 上创建仓库
2. 将代码推送到仓库
3. 在仓库设置中启用 GitHub Pages，选择 `dist` 目录作为源
4. 或者使用 GitHub Actions 自动部署

## 使用步骤

1. **上传Excel文件**：点击或拖拽企业微信导出的打卡记录Excel文件
2. **查看统计**：系统自动计算并显示：
   - 本月已上班天数
   - 本月剩余工作日
   - 本月已加班时长
   - 距离22小时还差多少小时
   - 平均每天需要加班多少小时
3. **查看补贴预估**：显示当前预估的补贴金额和补贴档位
4. **排除日期**（可选）：点击"排除日期"按钮，可以手动排除某些日期的打卡记录

## 加班规则

- 加班时间自19:00起算，半小时为单位计算
- 月累计加班达22小时以上开始享有加班补贴
- 补贴标准：
  - 22小时 ≤ 加班时长 < 44小时：15元/小时
  - 44小时 ≤ 加班时长 < 66小时：20元/小时
  - 66小时 ≤ 加班时长 < 90小时：23元/小时
  - 90小时以上：25元/小时，封顶2500元
- 加班超过20:00后公司提供晚餐补贴，20元/天

## Excel文件格式要求

Excel文件应包含两个Sheet：
1. **概况统计与打卡明细**（第一个Sheet）
2. **打卡详情**（第二个Sheet）

打卡详情Sheet应包含以下列：
- 日期（如：2025年9月30日 星期二）
- 打卡类型（筛选"下班"记录）
- 实际打卡时间（如：20:09）

## 技术栈

- Vue 3 (Composition API)
- Vite
- SheetJS (xlsx.js) - Excel解析
- 原生CSS（支持深色模式）

## 浏览器支持

- Chrome/Edge (推荐)
- Firefox
- Safari
- 移动端浏览器

## 注意事项

- 所有数据仅在浏览器本地处理，不会上传到任何服务器
- 数据保存在浏览器的localStorage中
- 建议定期导出Excel文件进行备份
